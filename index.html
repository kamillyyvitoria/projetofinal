Pytest


O que é?

O pytest é uma framework de teste pargita python,seus testes devem ser desenvolvidos em python, o que faz com que o código fique menor, seja mais legível e fácil de dar manutenção,tem por objetivo te permitir escrever testes pequenos de maneira fácil, e ainda assim ser possível escalar o seu uso para testes funcionais complexos.

Flask


O Flask é um micro framework(sequencia de passos para resolver um problema), que utiliza a linguagem Python para criar aplicativos Web. O framework flask vai resolver apenas o subconjunto limitado dos problemas clássicos na construção de aplicações Web, uma das razões para esse framework resolver apenas alguns problemas é que na realidade ele foi criado inicialmente como uma brincadeira de 1 de abril.


Problemas mínimos que o Flask resolve: Abstração do Protocolo HTTP, roteamento, Organização e Frameworks de Templates.
