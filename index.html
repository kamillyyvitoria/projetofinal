Git


O Git é um versionamento de arquivos, é um sistema de controle de versões distribuídos, usado principalmente no desenvolvimento de software. Existem três estados do git, sendo eles: consolidado, modificado e preparado.

GitHub


O GitHub é é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git, ele é considerado uma ferramenta essencial para engenheiros de software, é um serviço baseado em nuvem que hospeda um sistema de controle de versão, chamado Git. Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados.

Pytest


O que é?

O pytest é uma framework de teste para python,seus testes devem ser desenvolvidos em python, o que faz com que o código fique menor, seja mais legível e fácil de dar manutenção,tem por objetivo te permitir escrever testes pequenos de maneira fácil, e ainda assim ser possível escalar o seu uso para testes funcionais complexos. De código aberto, o PyTest é um framework de uso geral, mas é especialmente usado para testes funcionais e de API, gerando relatórios em diferentes formatos.


Como Instalar o pytest:

Antes de tudo o Python deve estar instalado na sua máquina (3.5 ou superior).Depois do Python já instalado ,é necessário também o Pip (pip é um gerenciador de pacotes para projetos em Python, utilizado para instalar, remover e atualizar pacotes nos nossos projetos). Conforme a instalação você deve digitar a seguinte instrução no Prompt de Comando: Pip install pytest.


Flask


O Flask é um micro framework(sequencia de passos para resolver um problema), que utiliza a linguagem Python para criar aplicativos Web. O framework flask vai resolver apenas o subconjunto limitado dos problemas clássicos na construção de aplicações Web, uma das razões para esse framework resolver apenas alguns problemas é que na realidade ele foi criado inicialmente como uma brincadeira de 1 de abril.


Problemas mínimos que o Flask resolve: Abstração do Protocolo HTTP, roteamento, Organização e Frameworks de Templates.

A maior vantagem do Flask é a sua flexibilidade.

Suas maiores desvantagens, são:Falta de ferramenta, Difícil entendimento e custos de manutenção.
